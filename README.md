- **Multi-Genome Analysis**: Batch processing of 1,032+ mitochondrial genomes from GenBank format
- **Gene Normalization**: Intelligent mapping of 50+ variant gene names to 13 standardized nomenclature
- **Product-Based Inference**: Rescue of CDS features lacking gene qualifiers through product annotation analysis
- **Comprehensive Statistics**: 
  - Nucleotide composition (A, T, G, C percentages)
  - GC/AT content ratios
  - GC skew analysis (strand bias detection)
  - Gene length distributions
- **Publication-Ready Visualizations**:
  - Averaged nucleotide composition heatmaps
  - AT/GC content comparison plots
  - Multi-panel compositional analysis
  - Clean, minimal figures for manuscripts
